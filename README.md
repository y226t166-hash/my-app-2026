# The Arcane Forge

『The Arcane Forge』は、ブラウザで遊べるダークファンタジー武器作りゲームです。
あなたは鍛冶師となり、リソースを収集し、伝説の武器を鍛え上げます。

**プロジェクトURL:** [https://y226t166-hash.github.io/my-app-2026/](https://y226t166-hash.github.io/my-app-2026/)

---

## 🎮 プレイヤーズガイド

### 1. 基本的なゲームの流れ
このゲームは、以下の3つのフェーズを繰り返すことで進行します。
1. **収集 (Gather):** 鉄、木材、マナを自らの手で集めます。
2. **鍛造 (Forge):** 材料を消費して、武器を製作します。
3. **管理・売却 (Inventory):** 武器を売却してゴールドを獲得します。

### 2. 各タブの解説
- **🛠 Gather:** ボタンをクリックしてリソース（Iron, Wood, Mana）を獲得します。
- **🔥 Forge:** レシピを選択して武器を鍛造します。強力な武器ほど時間がかかります。
- **📦 Inventory:** 製作した武器の性能（Power）とレアリティを確認し、売却できます。

### 3. レアリティとステータス
武器には以下の4つのレアリティがあり、性能は鍛造のたびにランダムに変化します。
- ⚪ **Common** / 🔵 **Rare** / 🟣 **Epic** / 🟠 **Legendary**

### 4. セーブデータについて
ブラウザの `localStorage` を使用して自動保存されます。ブラウザを閉じてもデータは保持されます。
