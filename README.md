# 神秘の鍛冶場 (The Arcane Forge) - 完全攻略マニュアル

![神秘の鍛冶場](./image.png)

『神秘の鍛冶場』へようこそ。本作は、伝説の武器職人を目指すブラウザベースのクリック＆マネジメントゲームです。
中世ファンタジーの世界でリソースを集め、お店を切り盛りし、王国からの依頼に応えて伝説の鍛冶聖を目指しましょう。

**プレイはこちら:** [GitHub Pages Project URL](https://y226t166-hash.github.io/my-app-2026/)

---

## 🧭 ゲームの目的
あなたの最終目標は、王国騎士団からの**【最終依頼】**を完遂し、**「伝説の鍛冶聖」**の称号を得ることです。

---

## 🏪 お店の経営 (Shop Management) **[NEW!]**
本作では、作った装備を自分の店で販売することができます。

### 1. 店名をつける
ヘッダーにある「店名変更」ボタンから、自分だけの鍛冶場の名前を設定できます。

### 2. 陳列棚に並べる
1. **鍛造**で装備を作成します。
2. **「倉庫 (Warehouse)」**タブを開きます。
3. 製作した装備のカードにある**「棚に並べる」**ボタンをクリックします。
4. 装備が**「店先 (Shop)」**の棚に移動します。

### 3. お客さんへの販売
- **「店先」**タブの棚に商品を並べておくと、時間の経過とともに冒険者や騎士が来店し、商品を購入していきます。
- **高利益**: 店先で販売すると、倉庫での即時売却よりも高い価格（威力の2.5倍）で売却できます。
- **来店履歴**: 誰が何を買っていったか、履歴で確認できます。

## 🛠 基本システム解説

### 1. リソース収集 (Gather)
- 💰 **ゴールド**: 従業員の雇用や高度なレシピに必要です。
- ⛓️ **鉄 / 🪵 木材 / 💎 マナ**: 装備の原材料です。クリックまたは雇用した職人が自動で集めます。

### 2. 鍛造 (Forge) & 一括作成
- **一括作成**: 数量を入力することで、一度に複数の装備を鍛造できます。
- **職人ボーナス**: まとめて作ることで、1個あたりの鍛造時間が短縮されます。

### 3. 装備の種類
- ⚔️ **武器**: 攻撃力に優れます。
- 🛡️ **盾・鎧**: 防御力に優れます。
- **レアリティ**: 一般(Common)から伝説(Legendary)まで存在します。

### 4. 依頼 (Quests)
- 王国からの特別な注文です。達成することで多額の報酬と称号を獲得し、ストーリーが進行します。

## 👥 鍛冶場の運営
稼いだゴールドで従業員を雇用し、素材収集を自動化しましょう。
- **技術研修**: あなた自身のクリック獲得量を増やします。
- **自動収集職人**: 採掘師、木こり、魔力探求者を雇うことができます。
